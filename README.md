# ReactJS Project

## 📋 Sobre

Projeto de frontend moderno usando React. Demonstra conhecimento de componentes reutilizáveis, gerenciamento de estado, hooks e boas práticas de desenvolvimento frontend com JavaScript moderno.

## 🛠️ Stack

- **React 18+** - Biblioteca UI
- **JavaScript ES6+** - Linguagem
- **CSS3** - Estilos
- **Vite** - Build tool (recomendado)
- **Node.js 16+** - Runtime

## 🚀 Como Começar

### Instalação
```bash
# Clone o repositório
git clone https://github.com/VGameleira/reactJS.git
cd reactJS

# Instale dependências
npm install

# Inicie desenvolvimento
npm run dev
```

### Build para Produção
```bash
npm run build
npm run preview
```

## 📁 Estrutura do Projeto

```
src/
├── components/        # Componentes reutilizáveis
├── pages/            # Páginas (rotas)
├── hooks/            # Hooks customizados
├── contexts/         # Context API
├── utils/            # Funções auxiliares
├── styles/           # Estilos CSS
├── App.jsx           # Componente raiz
└── main.jsx          # Entrada
```
